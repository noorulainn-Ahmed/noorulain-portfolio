## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid, Custom Variables)
- **Scripting:** Vanilla JavaScript (Intersection Observer API for animations, DOM manipulation)
- **Features:** 
  - Smooth scrolling navigation.
  - Responsive design for mobile and desktop.
  - Intersection Observer-based scroll animations.
  - Interactive contact form.

## 📂 Project Structure
- `index.html` - The main structure of the portfolio.
- `mainstyle.css` - Custom CSS for the minimalist aesthetic (Sage/Beige theme support).
- `script.js` - Logic for navigation, form handling, and scroll reveal animations.
